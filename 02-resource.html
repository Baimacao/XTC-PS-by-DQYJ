# 一、效力及构成

  • 本规范自【2022年7月27日】起生效，作为第三方应用与广东小天才科技有限公司（以下简称“小天才”）合作的前提，小天才有权对本规范进行修改或补充，该修改和补充在送达应用开发者后即时生效。
  • 除本规范直接规定的内容外，小天才可能就与第三方应用合作中涉及的各事项在本规范基础上进行额外或补充规定。相应规范，例如《小天才电话手表第三方应用内容未成年人保护规范》，与本规范一同构成第三方应用与小天才间合作的有效文件，自送达应用开发者后即时生效。
  • 本规范的生效基于中华人民共和国相关法律法规，本规范未尽内容，以相关法律法规规定为准。
  • 开发者违反本规范，小天才有权采取应用下架、审查、要求整改、要求说明、解除合作、要求赔偿等措施，开发者违反本规范导致的纠纷由应用开发者自行负责，因此导致小天才遭受的损失由应用开发者承担赔偿责任及消除影响义务。

# 二、原则

  • 符合法律法规： 第三方应用的功能、内容、付费模式、个人信息保护等各环节均应符合国家相关法律法规；
  • 响应国家未成年人保护行动： 第三方应用及应用运营者应当积极响应包括清朗行动在内的国家未成年人保护工作，遵守以下规范：
    • ① 尊重未成年人隐私权，严禁侵犯未成年人个人隐私，严禁实施或允许实施对未成年人人身攻击、侮辱谩骂等网络欺凌行为，严禁发布对未成年人造成不良导向的污言秽语。
    • ② 第三方应用严禁利用“网红儿童”牟利，严禁通过未成年人做出不雅姿势、性暗示动作吸引流量，严禁一切诱导或可能诱导未成年人采取危险行为的功能或内容；
    • ③ 第三方应用不得允许未满16周岁未成年人提供或变相提供网络主播服务，不得在小天才电话手表应用中针对任何内容提供打赏或变相打赏的功能。
    • ④ 第三方应用不得提供以未成年人为形象创作的色情小说、漫画内容，严禁引诱未成年人不良交友、煽动负面情绪。
    • ⑤ 第三方应用提供社交或类社交功能的，应当严厉查处对未成年人进行性引诱、教唆未成年人拍摄裸露视频图片、实施网络猥亵等行为。
    • ⑥ 第三方应用不得直接或允许应用使用者对散布虚假信息、虚构离奇故事和低俗剧情进行炒作引流、恶意营销等，对应用内涉及炫富拜金、奢靡享乐、扭曲审美等可能对未成年人造成不良影响的内容应当严格审查并禁止发布。
    • ⑦ 第三方应用不得提供网络游戏或网络游戏账号租售服务，不得以任何方式发布破解防沉迷系统的信息内容。
    • ⑧ 第三方应用应当遵守积极响应国家各类未成年人保护工作，例如由中央网信办开展的暑期清朗活动等；遵照相应未成年人保护工作的指导，对自身应用进行审查调整，提供适合未成年人使用的应用服务及绿色健康的互联网环境。
  • 个人信息安全： 第三方应用的用户个人信息须符合国家相关法律法规，遵循公开透明、方便管理、确保安全、细致管控、合理适度的处理原则；
  • 应用内容安全健康： 第三方应用须提供安全健康的网络环境，内容积极向上、阳光开朗，符合社会主义核心价值观。
  • 健康护眼： 第三方应用的功能应当符合国家近视卫生要求中关于用眼保护的原则，积极适配小天才电话手表的硬件特点，提供护眼健康的UI设计（包括但不限于颜色、字体、分辨率），避免直接或间接引导用户对手表屏幕进行重度使用；
  • 防止沉迷： 第三方应用应当注意未成年人防沉迷，应用功能设计须符合屏幕使用低粘度、轻互动的原则，避免游戏化设计；
  • 费用合理： 第三方应用的内容增值服务需物有所值，要为家长认同且对用户有价值；

# 三、应用安全

3.1基于应用安全，第三方应用开发者及其应用须设置以下必备功能及流程：

  • 应用需提供用户账号注销等能力。
  • 应用对用户个人信息的收集、存储、转移的机密性负责，避免外泄，禁止未经用户同意与其他应用共享、使用用户个人信息。
  • 应用须提供隐私政策说明，并按照现行法律向用户说明所收集或传输的用户数据内容以及数据的用途、分享对象和其他相关信息。
  • 当服务需要间接或者直接收集到用户数据的，必须事先获得用户的明确同意，同时应当告知用户相关数据收集的目的、范围及使用方式等，必须始终坚持用户知情权为前提。
  • 开发者需根据GBT 35273-2020《信息安全技术个人信息安全规范》进行信息安全保护。

3.2基于应用安全，第三方应用开发者及其应用有禁止事项如下：

  • 禁止含有病毒木马等危害用户的内容（包括恶意代码等可疑行为）。
  • 禁止在用户不知情或侵犯当前法律的情况下传输、收集或使用用户隐私数据，如通讯录、照片和短信记录等。
  • 禁止应用窃取用户密码或者其他用户个人数据。
  • 禁止超范围收集用户信息，即非服务所必需或无合理应用场景超范围或超频次收集个人信息。
  • 禁止“不给权限不让用”的行为，即向用户索取与当前服务场景无关的权限，用户拒绝后，应用退出或关闭。
  • 禁止用户在明确拒绝授权后，频繁申请与当前场景无关的录音、相机等权限。
  • 禁止过度索取权限，即在未使用相关功能和服务时提前开启权限或超出业务需要申请相录音、相机等权限。
  • 禁止未经用户授权强制启动系统服务（例如：蓝牙，网络服务等）;此外，若存在获取与软件无关的权限，有可能威胁到用户信息安全的行为，则不予通过。
  • 禁止阻碍用户设备正常通信等基础功能。
  • 禁止含有可执行代码的其他第三方应用或SDK。
  • 禁止含有非法监听、非法监控等非法行为。
  • 禁止未经用户同意或者强制要求用户同意，对用户个人信息、使用行为进行分析，为用户进行定向推送。
  • 禁止在用户不知情的情况下，超出业务正常需要，频繁联网消耗用户流量。

# 四、应用功能规范

4.1基于应用功能的规范化，第三方应用开发者及其应用须遵守以下规范：

  • 应用功能设计需以保护未成年人身心健康为标准，符合低粘度、轻互动的原则，避免手表屏幕重度使用，避免核心功能实现需要用户连续阅读或点击屏幕进行交互，禁止以视频、文字、图片浏览为主要功能。
  • 应用内不得含有堆箱子、投篮、射击、打斗、猜谜语、棋牌、成语消除、数独、2048等游戏或游戏元素。
  • 应用对于终端设备应具备良好的兼容性，需能正常安装、使用、卸载，运行时禁止出现频繁崩溃、卡顿等现象。
  • 应用安装后，禁止出现多个桌面快捷图标，应用图标、名称、实际功能等需与提交应用审核时填写的信息相符。
  • 应用功能需已完善并能正常使用，禁止上传功能体验不完整的版本。
  • 平台允许开发者根据运营需要，针对此应用所提供的服务，在应用内进行内容推广或广告促销（包括且不限于向用户宣传新功能/新内容、会员折扣活动等）。但是，平台禁止推广非此应用所提供的服务（包括且不限于实体书籍、文具等），以及其他任何形式的广告（相亲、贷款等）。 严禁以任何形式在应用内售卖任何实物，同时严禁通过应用内的二维码等形式为其导流。

4.2基于应用功能的规范化，第三方应用开发者及其应用有禁止事项如下：

  • 禁止应用内功能设计、互动方案、画面效果、场景音效、应用内容过于游戏化或含有过多游戏元素，包括但不限于答题闯关、竞技PK、地图探索、个性装扮、收集合成、宠物养成等。
  • 禁止未使用应用前强制登录第三方账号，应用需支持用户使用游客模式体验功能，且需支持用户在未登录的情况下能使用部分核心功能。
  • 禁止应用热更新（从开发者或其他第三方的网站下载可执行组件），或以其他方式使应用与提交到小天才审核（包括但不限于功能审核、内容审核等）的应用不一致，如更改应用付费渠道、更改应用界面、上线新功能等。
  • 禁止非功能必要时，刻意引导关注公众号、下载手机 APP 等导流行为。
  • 禁止影响终端设备的正常使用，包括但不限于：安装后修改系统配置导致用户无法修改、按键失灵等情况。
  • 禁止恶意影响其他应用的正常使用。
  • 禁止强制要求用户下载其他第三方应用。
  • 禁止含有隐藏或不被用户感知的功能。
  • 禁止造成终端设备不正常的过度发热和耗电过快。

# 五、应用内容规范

5.1应用内容需符合《中华人民共和国未成年人保护法》《中华人民共和国预防未成年人犯罪法》《中华人民共和国广告法》《未成年人节目管理规定》《互联网视听节目服务管理规定》等法律法规要求；

5.2基于应用内容的规范化，第三方应用开发者及其应用须遵守以下规范：

  • 应用内容应符合《小天才电话手表第三方应用内容未成年人保护规范》；
  • 培育积极健康、向上向善的网络文化，禁止任何不适合未成年人接触的内容流向电话手表用户。
  • APP需提供相应资质证明，根据APP分类不同，需要提供的资质各不相同，请查看附件1《应用服务资质清单参考》，并建立内容审核机制。应用上架前需提供内容审核机制说明、内容审核报告、内容测试报告。
  • 应用内容更新前需将内容清单或审核报告提交平台，经平台确认后方可上线新内容。
  • 需具备过滤违规内容、服务关闭、对重度违规用户停止服务等管控能力，有效管制用户产生的内容。

5.3基于应用内容的规范化，第三方应用开发者及其应用有禁止事项如下：

  • 禁止散布淫秽、色情、赌博、暴力、凶杀、恐怖或者教唆犯罪的内容。严管不良传教等包装成课程产品、文化交流等的形式存在。
  • 禁止上线网络小说类、电台类等长音频（eg.斗罗大陆、吞噬苍穹；情感电台等）。
  • 禁止渲染校园恋情、校园霸凌、拜金主义、封建迷信等传达不良价值观的内容呈现。
  • 禁止“蓝鲸死亡游戏”、“死亡窒息”、“心跳文学”等诱导青少年自杀倾向的内容。
  • 禁止无底线追星、饭圈互撕等价值导向不良价值观内容出现，严厉打击与未成年人对于平台热搜打榜应援、明星话题刷屏、挑拨未成年人对立、对骂互喷、人肉搜索、攀比炫富等相关的内容。
  • 禁止应用内出现劣迹艺人、敏感艺人的作品、信息及所有可能对未成年人产生不良影响的内容。
  • 禁止传达错误价值观的内容呈现，如仅以偷、抢等传递错误价值的互动方式，例-偷好友能量碎片；如出现游戏、对战、欺负（好友）等非正向理念传达的字眼。
  • 禁止应用素材具备诱导性，如游戏性强、诱导下载的应用命名（如 xx 射击、 xx 大作战）、应用图标等，可结合应用核心服务及应用特点展开（如 xx 互动英语）。
  • 禁止应用内容是简单打包的内容聚合、页面整合。
  • 禁止提交大量内容相似的应用，或提交的应用与其他开发者应用相同或过分相似。
  • 禁止提供有明确版权归属的破解版、汉化版。
  • 禁止带有修改其他应用数据、存档等内容的功能。
  • 禁止含有反对宪法所确定的基本原则的内容。
  • 禁止含有危害国家安全、泄露国家秘密、颠覆国家政权、破坏国家统一的内容。
  • 禁止含有损害国家荣誉和利益的内容。
  • 禁止含有煽动民族仇恨、民族歧视，破坏民族团结的内容。
  • 禁止含有破坏国家宗教政策、宣扬邪教和封建迷信的内容。
  • 禁止含有散布谣言、扰乱社会秩破坏社会稳定的内容。
  • 禁止含有侮辱或者诽谤他人，侵害他人合法权益的内容。
  • 禁止含有法律、行政法规禁止的其他内容。
  • 禁止用不具有识别性的词汇来命名应用，如归纳类应用名称。
  • 禁止强制、误导、诱导用户点击广告，或者强制安装软件、捆绑下载。

# 六、应用付费规范

6.1基于应用付费内容及流程的规范化，第三方应用开发者及其应用须遵守以下规范：

  • 应用必须接入小天才付费通知、付费管控、账单管理、退款流程，保证家长对付费的知情与管控。
  • 应用付款页面需依照小天才规范设计，具体要求如下： (不得使用强烈的付费引导提示、促使用户前往付费，不得通过“喊爸爸妈妈扫码领取”等语音、文字诱导未成年人要求家长付费； (应用需增加明显的付费提醒，告知用户“该内容为付费内容，必须经过家长同意才能开通”，用户在提示界面点击确认提示后，应用方可展示付费服务SKU页面；
  • 应用应根据提供的服务合理定价。
  • 因儿童自行付费导致监护人需要退款的，在接收到用户反馈退款的1个工作日内将用户反馈告知小天才，由小天才为用户处理退款。
  • 按照时长购买的服务，累计购买会员时长不得超过1年。
  • 付费服务需明确告诉用户付费后可享受的服务，明示内容需保证真实准确、醒目规范，经用户二次确认后方可扣费，付费后获取的服务需与承诺一致。
  • 若因未明示用户付费后可享受的服务导致用户投诉，开发者必须按相关政策退款并承担相应责任。
  • 付费应用必须包含免费体验内容，要求如下：
类别 免费使用要求
工具类 每天需有固定的免费使用次数（具体次数与平台协商确定）。
内容类 免费内容占比不低于总内容的30%，且首页需有免费专区（不包括限免、等免等），内容更新后需保证免费内容占比不低于30%。

6.2基于应用付费内容及流程的规范化，第三方应用开发者及其应用有禁止事项如下：

  • 禁止出现未经用户二次确认主动扣费、隐形扣费或误导、诱导用户付费等恶意吸费的行为。
  • 禁止手表端应用内的任何广告BANNER、弹窗及营销活动，但通过应用BANNER对应用内的内容进行推荐的，如内容上新推荐、本周故事推荐等不受前述限制。
  • 禁止实施价格欺诈。
  • 禁止通过价格实施国家法律所禁止的不正当竞争行为。

# 七、健康使用规范

7.1基于应用健康使用规范，第三方应用开发者及其应用须遵守以下规范：

  • 开发者应提供合理的健康使用管控项；
  • 开发者应对应用的使用时长进行限制，到达使用限额时，用户无法继续正常使用功能的，可提示用户休息或呈现其他合适的提示语。
  • 基于产品屏幕及功耗特性，为儿童用户更好的使用产品服务，应用使用时长需遵循以下规范：
类别 使用时长限制
非音频类（不可后台播放音频的App） 单次前台连续使用不得操作10min，1小时内总计使用时间不超过15分钟，超过后单次休息时长不少于 10min；
音频类（可后台播放音频的App） 单次前、后台使用超过30min，需暂停音频播放并提示休息 10min，用户可自行选择是否继续播放音频。
工具类 不限制
