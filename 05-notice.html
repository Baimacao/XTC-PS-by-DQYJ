# 3. 关于投诉通知

若发现小天才应用市场上传的其他应用侵犯您的合法权益，可按侵权投诉流程发送投诉通知，投诉通知需下载投诉通知书模板填写包括但不限于以下内容：

# 3.1 投诉方信息

  • 个人

    姓名、住址、身份证明（中国大陆居民身份证、外国人护照、港澳台地区回乡证等）、联系方式（电话及邮箱）。如为代理人需提供相关授权证明材料。

  • 企业

    企业名称、地址、营业执照复印件、联系方式（电话及邮箱），如为代理人需要提供相关授权证明材料。

# 3.2 投诉简介

  • 侵权的应用名、内容网址等。

  • 侵权事实的客观描述。

  • 投诉通知要求：指投诉方要求小天才应用市场对被投诉方提供的内容、应用采取何种措施，如删除作品、断开链接等。

# 3.3 佐证侵权的证明材料

  • 权属证明：包括但不限于对作品享有版权或依法享有信息网络传播权的权属证明、相关有权机构颁发的著作权证书、专利权证书、商标权证书、作品备案证书、作品首次公开发表或发行日期证明材料、创作手稿、经权威机构签发的作品创作时间戳等能证明权利归属的有效材料。

  • 被投诉者构成侵权的证明：对涉嫌侵权作品侵权事实的举证包括但不限于有效证明材料、书面事实说明或法院判决书等。

# 3.4 投诉方保证声明

承诺投诉通知中所提供的材料和相关陈述真实、有效、合法，并保证由于投诉方通知及相关材料、文件引起的一切后果皆由投诉方承担。投诉方保证承担和赔偿小天才应用市场根据投诉通知要求下架有关涉嫌侵权的内容、应用而对小天才应用市场造成的任何损失。

# 4. 判断投诉通知有效性和转发投诉

  • 投诉方未按照本条内容进行投诉通知的视为未发出通知。

  • 投诉方明显恶意通知将不予受理，如已受理并造成小天才应用市场的损失，应当加倍承担赔偿责任。

  • 小天才应用市场在收到符合要求的投诉通知后，会根据法律要求处理并转送至被投诉方。

# 5. 关于投诉反通知

若被投诉方不认可小天才应用市场转送的投诉通知及处理措施，应在不超过3个工作日期限内向小天才应用市场发起侵权反通知，需下载投诉反通知书模板填写包含但不限于以下内容：

# 5.1 被投诉方信息

  • 个人

    姓名、住址、身份证明（中国大陆居民身份证、外国人护照、港澳台地区回乡证等）、联系方式（电话及邮箱）。如为代理人需提供相关授权证明等证明材料。

  • 企业

    企业名称、地址、营业执照复印件、联系方式（电话及邮箱），如为代理人需要提供相关授权证明材料。

# 5.2 反通知简介

  • 被投诉的侵权应用名、内容网址等。

  • 不认可投诉的理由：包括被投诉方主张享有的权利、投诉方的证明材料问题、法律依据等。

  • 反通知要求：指被投诉方向小天才应用市场提出的相关内容处理要求，如恢复被删除作品或恢复链接。

# 5.3 佐证未侵权的证明材料

  • 权属证明：包括但不限于相关有权机构颁发的著作权证书、专利权证书、商标权证书、作品备案证书、作品首次公开发表或发行日期证明材料、创作手稿、经权威机构签发的作品创作时间戳等能证明权利归属的有效材料；若是针对其他权利侵权导致删除（名誉权、荣誉权、隐私权等），应提供相应的证明材料。

  • 未侵权证明：对作品未侵权事实的举证包括但不限于未侵害投诉方权益的的有效证明材料、书面事实说明或法院判决书等。

# 5.4 被投诉方保证声明

承诺反投诉通知中所提供的材料和相关陈述真实、有效、合法，并保证承担和赔偿因根据被投诉方的反通知而恢复有关内容、应用的链接而给小天才应用市场造成的任何损失。

# 6. 反通知处理

  • 被投诉方未按照本条内容进行反通知的视为未发出反通知。

  • 被投诉方明显恶意反通知将不予受理，如已受理并造成小天才应用市场的损失，应当加倍承担赔偿责任。

  • 小天才应用市场在收到符合要求的反通知后，会根据法律要求处理并转送至投诉方，并同时告知投诉方可以向主管部门投诉或者向人民法院起诉。

  • 对于多次被投诉侵权，且不提供有效反通知或者反通知的内容不能初步证明其提供的内容不构成侵权的开发者，小天才应用市场有权停止相关平台服务。此外，根据侵权情节、投诉方要求等具体情况，小天才有权依单方判断将前述投诉反馈时限缩短为24小时。

# 7. 注意事项

  • 为了确保投诉的真实、有效，投诉证明材料和反诉举证材料中涉及的相关资质证明，请优先提供原件，若不能提供原件，请提供权利人签字盖章的复印件；电子复印件请通过官方受理邮箱发送。

  • 若投诉方已经向行政部门投诉或向法院起诉被投诉方，请将受理证明、提交的证据材料以及行政机关或法院的决定或判决一并提交至小天才应用市场，辅助争议内容做有效判断。
